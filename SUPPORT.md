# Support

iPhoneMirror is a community-maintained public preview.

## Before requesting help

1. Install Apple Devices or Apple Mobile Device Support.
2. Unlock the iPhone, reconnect it and confirm “Trust This Computer”.
3. Use the latest release.
4. Check the in-app live log and the driver state shown for the selected phone.
5. Read the [README troubleshooting and driver notes](README.md).

For reproducible bugs, use the repository's Bug Report form. For feature ideas,
use the Feature Request form.

Sanitize all attachments. Remove UDIDs, device names, user paths, account data,
pairing records and personal screen content. Do not post raw USB captures.

Security vulnerabilities must be reported privately through
[GitHub Security Advisories](https://github.com/RayrenSX/iPhoneMirror/security/advisories/new).
