# iPhoneMirror 0.3.0 Preview 1

首个可下载的 Windows x64 公开预览版。

## 重点功能

- USB 有线 iPhone 画面与 48 kHz 双声道系统音频采集。
- Media Foundation H.264 解码与 D3D11/DirectComposition 低延迟预览。
- 多设备发现、刷新和安全切换采集会话。
- 本地分辨率/FPS 限制、音量、静音、旋转、全屏、截图与实时日志。
- 独立预览和 OBS Window Capture 专用窗口。
- 按 iPhone 设备实例安装 libusb0 UpperFilter，包含文件校验和失败回滚。
- 简体中文与英文界面。

## 使用要求

- Windows 10 19041+ 或 Windows 11，x64。
- Apple Devices 或 Apple Mobile Device Support。
- 数据线连接、手机解锁并信任电脑。

下载 `iPhoneMirror-v0.3.0-preview.1-win-x64.zip`，校验 `SHA256SUMS.txt`
后解压，运行 `iPhoneMirror.exe`。应用为 .NET 自包含发布。

## 兼容性验证

- iPhone18,3 / iOS 26.5.2：1206×2622，约 58.6 FPS。
- iPhone13,1 / iOS 18.7.8：1082×2340，约 58.9 FPS。

## 预览版说明

- 程序和提权 Helper 尚未商业签名，SmartScreen/UAC 可能显示未知发布者。
- 每台 iPhone 第一次使用时都需要单独安装过滤器并可能需要重插。
- 禁止使用 Zadig 替换 Apple 父设备驱动。
- OBS 当前通过 Window Capture 输出，不包含虚拟摄像头。
- Apple Screen Capture 是私有协议，未来 iOS 更新可能需要适配。
