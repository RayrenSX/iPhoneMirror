# iPhoneMirror v1.0.3

正式版，面向 Windows 10/11 x64。

## 本版重点

- USB 有线投屏与局域网 AirPlay 无线投屏继续共用多设备会话、预览、音频、截图、独立窗口和 OBS 工作流。
- 无线 AirPlay 接收端默认启动，并支持连接前选择最大质量、1080p、720p 或 540p 广播规格。
- 有线投屏保留 A 演示模式、B AirPlay 实验模式和 C 爱思兼容模式；高级 USB 尺寸设置仍只对 B 模式开放。
- 驱动管理器独立运行，支持 Apple USB 环境检测、libusb 过滤驱动安装、修复、卸载、日志和设备状态诊断。
- 多设备列表支持长按排序、独立预览、单窗口静音及静音其他窗口。
- 无线设备握手元数据会尽量解析为可读的 iPhone/iPad 型号，而不是直接显示 ProductType 编号。
- 应用提示窗口、驱动管理器和中英文资源保持统一 UI 与语言切换行为。
- 预览切换使用原生窗口同步和过渡遮罩，减少设备切换时白闪、边框闪动和旧画面残留。
- 教程图片暂不随本版本发布，文字教程保留完整功能说明，后续可独立补充截图。

## 构建与验证

- Native Core protocol tests: passed
- WirelessHost smoke test: passed
- App logic tests: passed
- Driver installer tests: passed
- Release target: `win-x64`

## 已知限制

- Apple USB 与 AirPlay 使用私有协议，未来 iOS 版本可能改变行为。
- OBS 当前推荐使用独立预览窗口进行 Window Capture，虚拟摄像头仍未内置。
- Windows Authenticode 签名和干净系统矩阵验证仍是后续工作。
