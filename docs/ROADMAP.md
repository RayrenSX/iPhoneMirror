# 后续升级计划

本文件只记录计划，不代表当前版本已经实现，也不作为当前版本的功能承诺。

## 1. 稳定性与兼容性

- 建立 Windows 10/11、Apple Devices、不同 iOS 版本和 USB 芯片组合的干净环境测试矩阵。
- 增加断线、锁屏、拔线、快速重连和多设备同时停止的自动化回归测试。
- 为驱动安装、修复和卸载补充失败回滚、权限异常和重启后恢复测试。

## 2. 无线 AirPlay

- 增加网络带宽、丢包、解码队列和实际帧率监控。
- 根据网络状态提供可选的自适应清晰度建议，不自动改变用户已经确认的广播规格。
- 完善 iPhone、iPad 和不同系统版本的握手型号映射，并为未知 ProductType 提供可维护的本地映射表。
- 评估 IPv6、企业 Wi-Fi 隔离和多网卡环境下的发现与连接策略。

## 3. 有线投屏与画质

- 继续验证 A/B/C 模式在新 iOS 版本上的画面输出和视频 App 行为。
- 为高级 USB 尺寸提供安全预设、恢复默认值和失败后的自动回退。
- 增加渲染帧率、解码帧率和源设备分辨率的分层诊断，避免把本地渲染限制误认为源码流降质。

## 4. 采集与创作工具

- 评估虚拟摄像头输出，保持独立窗口采集作为稳定回退方案。
- 增加截图格式、保存位置和连续截图策略，同时继续保证截图不包含软件 UI。
- 优化多设备独立预览的布局、窗口记忆和 OBS 场景配置辅助。

## 5. 发布工程

- 接入 Windows Authenticode 签名和发布包完整性校验。
- 在 GitHub Actions 中运行 native、应用逻辑、驱动管理器和打包测试。
- 生成可复现的 SBOM、第三方二进制校验和发布资产清单。
- 增加升级说明模板和回滚说明，区分正式版、预览版与驱动组件版本。

## 6. 用户体验

- 完成无障碍名称、键盘导航和高 DPI 检查。
- 统一所有错误、确认、权限和重连提示的可操作建议。
- 教程截图由维护者逐张补充，所有截图在公开发布前移除 UDID、序列号、通知和日志中的个人信息。
